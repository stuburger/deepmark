00 · Principles
Anti vibe-code principles
DeepMark is a professional tool for teachers — not a consumer app. Every decision should read as intentional.
Never use these
Inter — the most common AI-generated UI font. Replaced with Geist everywhere.
Pill radius (20px+) — everywhere in consumer apps. All radii are 5px maximum. No exceptions.
Soft brand-coloured glows — all shadows are hard SE offset in black only.
Purple #6B4FA0 — the AI default colour of 2025–26. Replaced with teal #01ADD0.
Uniform stroke weight icons — Lucide/Heroicons default. Use the custom DeepMark set.
Design intent
Brand identity is carried by typography and texture — not colour fills. Teal appears as punctuation only. The paper texture and Playfair Display greeting do the identity work. Everything else should feel like a serious professional tool.

03 · Spacing & Radius
Border radius — strict
5px is the universal radius. No exceptions except the floating toolbar.
Hard rule
No element may have border-radius above 10px. Pills (20px+) are strictly forbidden. If you write border-radius above 10px — stop.
--radius-sm · 5px
border-radius: 5px
Buttons · inputs · badges · chips · cards · all UI. Universal default.
--radius-md · 8px
border-radius: 8px
Upload zones · callout panels · expandable feedback panels.
--radius-lg · 10px
border-radius: 10px
Dialogs · modals · floating toolbar only. Nothing else.

08 · Architecture
Surface taxonomy
Agreed with Stu Bourhill. Every screen must be one of these four types — determines routing, mounting behaviour, and parent lifecycle.
Page (URL route)
Has its own IA below it. Sub-routes, nested state, child pages. Parent unmounts on navigate.
Dashboard · All papers · Analytics
URL-state dialog
Focused destination, parent stays mounted. Linkable via nuqs query param (?job=…). Survives refresh.
Script viewer (?script=...) · Marking job (?job=...)
Plain dialog
Short, scoped, ≤2 steps. Fully reversible by closing. Not linkable.
Mark new paper · Grade boundaries · Delete confirm
Sheet / Drawer
Deep secondary context, user keeps their place. Slides over content.
Sidebar nav · AO breakdown panel
